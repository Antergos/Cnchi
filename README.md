# Cnchi

Graphical Installer for Cinnarch Linux

## Dependencies

 * gtk 3
 * python 3
 * python-gobject 3
 * python-dbus
 * python-cairo
 * libtimezonemap
 * webkitgtk3
 * parted (dosfstools, mtools, ntfs-3g, ntfsprogs)
 * py3parted (pyparted on python3) -> https://github.com/cinnarch/cinnarch-packages/tree/master/py3parted
 * pacman
 * pyalpm
 * hwinfo
 * hdparm
